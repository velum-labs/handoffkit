# @fusionkit/plane

Legacy governance stack (not part of the FusionKit product).

Legacy governance control plane for contracts, policy, approvals, receipts, secret brokering, audit export, and the control-panel UI.

## Architecture

This package belongs to the legacy governance stack, not the FusionKit model-fusion product. It remains published from this monorepo while separation work is tracked, but the product path starts at `@fusionkit/cli` and the model-fusion gateway.

## Usage

New FusionKit users probably want `@fusionkit/cli`, not this package. See [../../docs/scope.md](../../docs/scope.md) for the product vs. legacy package map.
