## Purpose

Measure what happened, triage what users reported, and decide the next bet.

## Done when

- [ ] Metrics dictionary filled for every required metric

- [ ] Review sections filled (what worked / failed / wrong assumptions)
- [ ] Decision is exactly one of: Continue / Narrow / Reposition / Stop
- [ ] Both founders signed
- [ ] Blueprint follow-ups listed (for the next launch’s template)
