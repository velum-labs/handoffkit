# velum-model-fusion-protocol

Python distribution for the Model Fusion protocol bundle.

The package includes:

- JSON Schema durable record contracts
- OpenAPI 3.1 HTTP/JSON API contract
- protocol package metadata with the schema bundle hash

It is intended for private PyPI-compatible registries such as Cloudsmith, AWS
CodeArtifact, or Gemfury. Do not publish this package to public PyPI unless the
protocol governance changes.
