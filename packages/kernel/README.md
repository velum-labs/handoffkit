# @fusionkit/kernel

Typed runtime-kernel substrate for FusionKit workflows.

## Architecture

This package provides artifacts, operators, graphs, schedulers, budgets, traces, outcomes, streaming, and wire artifacts for model-fusion workflows.

## Usage

Use it for advanced workflow construction; most users should start with `@fusionkit/cli` or `@fusionkit/ensemble`.

```ts
import * as fusionkitPackage from "@fusionkit/kernel";
```
