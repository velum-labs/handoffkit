# @fusionkit/example-utils

Private shared helpers for the runnable FusionKit examples (never published).

## Architecture

This package provides demo manifest parsing (`examples/manifest.json`), per-demo narration banners, mock model helpers, and live-model configuration used by the `examples/*` projects and `scripts/demo.mjs`.

- Product docs: https://fusionkit.velum-labs.com (see the Examples guide)
- Example registry: [../../examples/manifest.json](../../examples/manifest.json)
