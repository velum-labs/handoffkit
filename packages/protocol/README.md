# @fusionkit/protocol

Shared FusionKit protocol contracts and validators.

## Architecture

This package contains zero-dependency schema helpers, hashes, traces, receipt primitives, and generated model-fusion validators used across the Node packages.

## Usage

Use it when validating model-fusion records or normalizing trajectories outside the CLI.

```ts
import * as fusionkitPackage from "@fusionkit/protocol";
```
