## The `.fusionkit/` folder

Fusion settings live in a committed `.fusionkit/` folder at your repo root. It is **safe to commit**. It stores only the env-var *names* that hold API keys (`keyEnv`), never the secret values.

```
.fusionkit/
  fusion.json
  prompts/
    judge.md
    synthesizer.md
```

Scaffold it interactively:

```bash
fusionkit init
```

Inspect the effective configuration and a dry-run preview any time:

```bash
fusionkit status
fusionkit config show
```

## Precedence

```
explicit CLI flags  >  .fusionkit/fusion.json  >  built-in defaults
```

The file is a default layer, not a lock. Any flag overrides it for one run.

## Settings (`fusion.json`)

```json
{
  "version": "fusionkit.fusion.v3",
  "tool": "codex",
  "defaultEnsemble": "default",
  "ensembles": {
    "default": {
      "panel": [
        { "id": "gpt", "model": "gpt-5.5", "provider": "openai", "keyEnv": "OPENAI_API_KEY" },
        { "id": "sonnet", "model": "claude-sonnet-4-6", "provider": "anthropic", "keyEnv": "ANTHROPIC_API_KEY" },
        { "id": "gemini", "model": "gemini-2.5-pro", "provider": "google", "keyEnv": "GEMINI_API_KEY" }
      ],
      "judgeModel": "gpt-5.5",
      "synthesizerModel": "gpt-5.5"
    }
  },
  "local": false,
  "observe": false,
  "onRateLimit": "fusion",
  "budgetUsd": 5,
  "portless": true,
  "port": null
}
```

### Fields

| Field | Type | Notes |
| --- | --- | --- |
| `version` | string | Must be `"fusionkit.fusion.v3"`. v1/v2 files are accepted and upgraded in memory. |
| `tool` | `codex` \| `claude` \| `cursor` \| `serve` | Default agent when none is passed. |
| `ensembles` | object | Named panels. The `default` ensemble advertises `fusion-panel`; other names advertise `fusion-<name>`. |
| `defaultEnsemble` | string | Which ensemble new sessions use by default. |
| `local` | boolean | Use the local MLX trio by default. |
| `observe` | boolean | Boot the observability dashboard by default. |
| `onRateLimit` | `fusion` \| `passthrough` \| `fail` | Vendor rate-limit behavior. Defaults to `fusion`. |
| `budgetUsd` | number | Optional session spend cap in gateway-observed USD. |
| `portless` | boolean | Use stable portless URLs when available. Defaults to `true`. |
| `port` | number \| null | Fixed gateway port, otherwise ephemeral. |

### Ensemble fields

| Field | Type | Notes |
| --- | --- | --- |
| `panel` | array | Panel members. Omit it on `default` to use the built-in panel. |
| `judgeModel` | string | Model used for judge synthesis. Defaults to the first panel model. |
| `synthesizerModel` | string | Optional final-output model. Defaults to the judge on the Python side. |

### Panel entries

| Field | Type | Notes |
| --- | --- | --- |
| `id` | string | Stable identifier for the panel member. |
| `model` | string | The model name. |
| `provider` | `mlx` \| `openai` \| `anthropic` \| `google` \| `openrouter` \| `openai-compatible` | Defaults to `mlx`. |
| `keyEnv` | string | Env var holding this model's API key. |
| `baseUrl` | string | Optional custom base URL. |

An `openrouter` member uses OpenRouter's `vendor/model` id and defaults to `OPENROUTER_API_KEY`:

```json
{ "id": "kimi", "model": "moonshotai/kimi-k2-thinking", "provider": "openrouter", "keyEnv": "OPENROUTER_API_KEY" }
```

## Edit from the CLI

```bash
fusionkit config get budgetUsd
fusionkit config set budgetUsd 5
fusionkit config unset budgetUsd
fusionkit config edit
fusionkit ensemble list
fusionkit ensemble add fast --model kimi=openrouter:moonshotai/kimi-k2-thinking
fusionkit ensemble use fast
```

## Export router YAML

The Node CLI derives the Python router YAML from `.fusionkit/fusion.json`:

```bash
fusionkit config export-yaml
fusionkit config export-yaml -o router.yaml
fusionkit serve --config router.yaml
```

## Prompt overrides (`prompts/`)

Each `.fusionkit/prompts/<id>.md` file overrides one built-in system prompt used by the synthesizer. A file that exists and is non-empty replaces the default; an empty or missing file falls back to the built-in.

```bash
uvx fusionkit prompts dump --dir .fusionkit/prompts
fusionkit prompts list
fusionkit prompts edit synthesizer
fusionkit prompts reset synthesizer
```

<Callout type="warn">
  Never put API key values in `.fusionkit/`. Store only the env-var name in
  `keyEnv` and export the secret or put it in `.env`.
</Callout>
