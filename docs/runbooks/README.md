# RouteKit incident runbooks

These runbooks are the operational response layer for RouteKit. Security
reports enter through the private channel in [`SECURITY.md`](../../SECURITY.md).

- [Credential compromise](credential-compromise.md)
- [Bad release](bad-release.md)
- [Provider outage](provider-outage.md)

Each incident must have one owner, timestamps, preserved evidence, a customer
impact statement, and explicit recovery validation.
