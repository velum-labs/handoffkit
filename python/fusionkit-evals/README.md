# fusionkit-evals

Evaluation and Pareto analysis helpers for FusionKit.

This package contains benchmark runners, tiny fixtures, public-benchmark tooling, prompt tuning, and score analysis used to measure fused panels against individual models. It is maintainer-oriented but ships with the Python package set today.

Most users should start with `@fusionkit/cli`; install evaluation extras only when running benchmarks.

Docs: https://fusionkit.velum-labs.com
Repository: https://github.com/velum-labs/handoffkit
