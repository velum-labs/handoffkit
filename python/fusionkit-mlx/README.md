# fusionkit-mlx

Optional MLX helper utilities for FusionKit local-model workflows.

This package contains Python-side helpers for Apple-Silicon local model paths. The product flow is exposed through `@fusionkit/cli` commands such as `fusionkit models` and `fusionkit codex --local`.

Most users should install `@fusionkit/cli`; install MLX extras only on supported macOS systems when running local panels.

Docs: https://fusionkit.velum-labs.com
