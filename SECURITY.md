# Security Policy

HandoffKit is currently a private, design-stage repository. Please do not open
public issues or discussions for suspected vulnerabilities, leaked credentials,
or sensitive design concerns.

## Reporting a Vulnerability

Report security concerns privately through GitHub's private vulnerability
reporting flow for this repository when available. If that flow is unavailable,
contact Velum Labs through your established private communication channel and
include `SECURITY` in the subject or opening line.

Please include:

- a concise description of the issue;
- affected files, workflows, or configuration;
- reproduction steps or evidence, if available;
- whether any secret, token, customer data, or private design material may be
  exposed.

Do not include live secrets in the report. If a secret may have been exposed,
identify the credential type and location so it can be rotated through the
appropriate private channel.
